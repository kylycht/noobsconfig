The files must be extracted to the Raspberry Pi's SD-Card.
In windows, you should have a single folder on the SD-Card called "network" which will contian the files.

On the Raspberry Pi, the files should be located at "/boot/network".

Default direct IP address is 169.254.0.10

Ensure you update the cmdline.internet file if you intend to use this.
